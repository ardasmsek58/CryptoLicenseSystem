Detailed Description

The LicenseGenerator class.

Provides functionality to generate a license file for a specific hardware ID. The license file includes the hardware ID and a digital signature generated using the provided RSA private key. The output is saved in JSON format.

static

Generates a license file for the given hardware ID.

Generates a license file for a given hardware ID using RSA private key signing.

This function creates a JSON license file containing:

  • The hardware ID of the target machine
  • A digital signature (SHA256 with RSA) of the hardware ID
Parameters
hardwareIdThe hardware fingerprint or ID to license.
privateKeyPathPath to the RSA private key file (private_key.pem) used for signing.
outputFilePath to save the generated license file (license.lic).
Returns
true if license generation is successful, false otherwise.

The function:

  1. Creates a JSON object containing the hardware ID
  2. Signs the hardware ID using SHA256 with RSA private key
  3. Saves the license as a .lic file containing both hardware ID and signature
Parameters
hardwareIdThe hardware fingerprint or ID for the target machine.
privateKeyPathPath to the RSA private key (private_key.pem) used for signing.
outputFilePath to save the generated license file (license.lic).
Returns
true if license generation succeeds, false otherwise.
